## 维度 4+5：产业趋势与冷思考

### 4.1 行业生命周期
- 导入期 / 成长期 / 成熟期 / 衰退期
- 搜索：[行业] + "行业报告"
- **必须引用第三方数据源**（券商报告/行业协会白皮书/第三方研究报告），不能直接用 BP 数据

### 4.2 市场空间修正
- **BP 声称 vs 第三方数据逐条对比**
- 搜索：[行业/产品] + 市场规模 / TAM
- 偏差 >20% 必须标注
- **必须多个来源交叉验证**

### 4.3 增长驱动力 + 失败案例
- 政策补贴 vs 真实降本增效
- 行业失败案例复盘（具体案例 + 涉案金额 + 失败原因）
- **必须列出实际案例，不泛泛**说

### 4.4 政策风险
- 政策变化 / 补贴退坡 / 监管
- **标注政策时间窗口**

## 维度 6：产品买家秀

### 6.1 BP 声称验证
- 提取 BP 中**所有**定量声称（出货量、订单金额、客户数、市占率等）
- 逐条验证
- **验证状态：✅通过 / ⚠️存疑 / ❌无证据 / 🔴矛盾**

### 6.2 招投标/政府采购验证
**必须搜的平台：**
- 中国政府采购网
- 各省市招投标平台
- 企查查/天眼查

### 6.3 客户反馈验证
- B 端：POC 测试报告/招投标公示/客户访谈
- C 端：用户评价/退货率/复购率

### 6.4 Red Flags
- BP 声称与第三方数据偏差 >20%
- 客户集中风险
- 产品交付能力存疑

## 维度 8：竞争格局分层分析

**必须分层搜索，禁止简单横向对比**

### Layer 1 - 直接竞品（同商业模式 + 同应用场景）
- 搜索：[产品功能] + 同类企业 + 融资/技术对比
- **必须列出至少 3 家直接竞品**

### Layer 2 - 替代威胁（不同模式但争夺同一客户预算）
- 搜索：[应用场景] + 替代方案 + 巨头布局

### Layer 3 - 产业链博弈
- 上游整合风险 / 下游自研趋势 / 跨界降维打击

### 竞争格局输出格式

| 企业名 | 竞争层级 | 商业模式 | 对比维度 | 威胁评估 | 数据来源 |

**禁止：**
- 用 IDM 企业营收直接对比 Fabless 企业
- 用上市公司财务数据对标 VC 阶段公司
- 忽略商业模式差异的简单对比
